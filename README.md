# NTUEEplus

#### elasticsearch database
Download [elasticsearch](https://www.elastic.co/guide/en/elasticsearch/reference/current/_installation.html) and open local database (localhost:9200)

```bash
$ elasticsearch/bin/elasticsearch
```

#### hot module
```bash
$ npm run start
```

#### production
```bash
$ npm run build
$ npm run start:prod
```

#### test database
See commands:
```bash
$ node manage.js --help
```
